# student_helper
تطبيق بسيط يساعد الطلاب على تنظيم واجباتهم الدراسية من خلال ادارة المهام اليوميه وتتبع مستوى الانجاز 
يدعم االغة العربية والانجليزية



A simple application that helps students organize their schoolwork by managing daily tasks and tracking progress
supports Arabic and English languages.
